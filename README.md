# Leaf&Cup — Чайный магазин

Веб-приложение чайного магазина.  
**Стек:** Node.js + Express.js + Sequelize + SQLite (бэкенд), HTML/CSS/JS (фронтенд).

## Структура проекта

```
leafcup/
├── server.js              ← точка входа (запуск сервера)
├── package.json
├── api/
│   ├── server.js          ← класс Server (Express, маршруты, БД)
│   ├── user.js            ← контроллер пользователей
│   ├── product.js         ← контроллер товаров
│   └── order.js           ← контроллер заказов
├── routes/
│   ├── Users.js           ← маршруты /api/user
│   ├── Products.js        ← маршруты /api/product
│   └── Orders.js          ← маршруты /api/order
├── database/
│   ├── database.js        ← подключение SQLite через Sequelize
│   ├── models.js          ← модели: User, Category, Product, Order, OrderItem
│   └── seed.js            ← начальные данные (товары, категории, администратор)
├── assets/                ← загружаемые изображения
└── client/                ← фронтенд (HTML/CSS/JS)
    ├── index.html
    ├── catalog.html
    ├── cart.html
    ├── account.html
    ├── admin.html
    ├── images/
    ├── styles/
    └── js/
        ├── api.js         ← запросы к серверу
        ├── auth.js        ← авторизация
        ├── cart.js        ← корзина
        └── app.js         ← утилиты
```

## Установка и запуск

### 1. Установить зависимости

```bash
npm install
```

### 2. Запустить сервер

```bash
node server.js
```

При первом запуске:
- автоматически создаётся файл базы данных `database/leafcup.db`
- заполняются категории и товары
- создаётся аккаунт администратора

### 3. Открыть сайт

Перейти в браузере: **http://localhost:8081**

## API-маршруты

### Пользователи `/api/user`
| Метод | URL              | Описание                    |
|-------|------------------|-----------------------------|
| POST  | /register        | Регистрация                 |
| POST  | /login           | Авторизация → JWT в cookie  |
| POST  | /logout          | Выход                       |
| GET   | /me              | Профиль текущего юзера      |
| PATCH | /me              | Обновить имя                |
| GET   | /all             | Все пользователи (admin)    |

### Товары `/api/product`
| Метод  | URL           | Описание                  |
|--------|---------------|---------------------------|
| GET    | /             | Список товаров            |
| GET    | /categories   | Список категорий          |
| GET    | /:id          | Один товар                |
| POST   | /             | Добавить товар (admin)    |
| PUT    | /:id          | Обновить товар (admin)    |
| DELETE | /:id          | Удалить товар (admin)     |

### Заказы `/api/order`
| Метод | URL          | Описание                   |
|-------|--------------|----------------------------|
| POST  | /            | Создать заказ              |
| GET   | /my          | Мои заказы                 |
| GET   | /all         | Все заказы (admin)         |
| PATCH | /:id/status  | Обновить статус (admin)    |
